# pdfmake browser assets

These files are the pinned browser distribution from `pdfmake` version `0.3.11`:

- `pdfmake.min.js`
- `vfs_fonts.js` (embedded Roboto font files)
- `poppins_vfs.js` (locally embedded Poppins family)
- `libre_baskerville_vfs.js` (locally embedded Libre Baskerville display family)

Source: <https://www.npmjs.com/package/pdfmake>

License. Their license files are stored with the source font files in
`fonts/`.
